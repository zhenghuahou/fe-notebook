方案一(元素不定宽高):
display:flex;
justify-content:center;
align-items:center;
方案二(元素不定宽高):
position:absolute;
left: 50%;
top: 50%;
transform: translate(-50%,-50%);
方案三(元素宽高固定):
如果子元素不设置宽度和高度，将会铺满整个父级 position:absolute;
top: 0;
bottom: 0;
left: 0;
right: 0;
margin: auto;
方案四(元素不定宽高):
父元素设置:
display: table-cell;
vertical-align: middle;
text-align: center;
子元素设置:
display: inline-block;(行内元素)
方案五(元素宽固定，高不固定):
父元素设置:
display: table-cell;
vertical-align: middle;
子元素设置:
margin: 0 auto;
width: 80%;(需要为width设置一个指，设置为固定的px值也可以)
